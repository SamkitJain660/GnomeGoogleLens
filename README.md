image search on google lens directly from the built in gnome screenshot tool

### Installation

To get started, you can install the extension using either of these methods:

* **Direct Download:** Install it through the official portal:
    [PENDING APPROVAL..](https://extensions.gnome.org)
* **Manual Method:** Copy the extension files into your local directory:
    `~/.local/share/gnome-shell/extensions/`
